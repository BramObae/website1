# MED AID Kenya — Health Accountability Platform
## Complete Source Code

### Files
- `MED_AID_Kenya_Complete_System.html` — **The full working system** (open this in any browser)
- `medaid_styles.css` — All CSS styles extracted separately
- `medaid_logic.js` — All JavaScript logic extracted separately

---

### How to Run
1. Download `MED_AID_Kenya_Complete_System.html`
2. Double-click it — opens in Chrome, Firefox, Edge, or Safari
3. No server, no internet, no installation needed

---

### Key Features
- File complaint (anonymous or identified)
- Upload evidence (photos, video, documents)
- Emergency escalation (red-flag routing)
- Unique tracking number per complaint
- Ministry posts official resolutions
- Community upvoting and commenting
- Hospital watchlist auto-built from complaints
- County-level breakdown (all 47 counties)
- Patient donation via M-Pesa
- Strike/demo/lobby notice board (timestamped)
- SMS/USSD access mentioned (*384#, SMS 21900)

---

### Technology Stack
- Pure HTML5, CSS3, JavaScript (ES6+)
- No frameworks, no dependencies, no server required
- Google Fonts (Playfair Display + DM Sans)

### To Deploy as a Real Website
1. Upload the HTML file to any web host (Hostinger, SiteGround, or free: GitHub Pages, Netlify)
2. For a database backend, connect to Firebase or Supabase
3. For real M-Pesa, integrate Safaricom Daraja API
4. For SMS/USSD, integrate Africa's Talking API
